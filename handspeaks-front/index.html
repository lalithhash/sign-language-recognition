Overview of the Project

How We Trained the Model



Connect to Bluetooth

Pair the smartwatch to the system to establish a connection.

Details:

We connected the smartwatch using Bluetooth through the Touch SDK. This established a seamless and stable connection, enabling efficient real-time data communication between the smartwatch and the system.

Collect Motion Sensor Data

Capture real-time motion data from the smartwatch sensors for training.

Details:

  • Acceleration: Capture 3D acceleration data (X, Y, Z) to measure changes in velocity.
  • Gravity: Track gravitational forces affecting the smartwatch for stability analysis.
  • Angular Velocity: Record rotational movements using gyroscopic data.
  • Orientation: Determine the smartwatch’s orientation using magnetometer readings and coordinate transformations.

All data is sent to the model for further processing and classification.

Preprocess the Data

Normalize and clean the raw data to ensure accuracy during training.

Details:

Remove noise, normalize sensor data, and prepare it for feature extraction. Ensure data quality for effective model training.

Train the Model

Use machine learning algorithms to classify and label the gestures.

Details:

We use an LSTM (Long Short-Term Memory) Recurrent Neural Network to process sequential motion data. The LSTM model efficiently captures temporal dependencies and learns patterns to accurately recognize specific gestures.

Test and Recognize

Perform gestures to validate the model's recognition capabilities.

Details:

Perform gestures to validate the model's performance. The system processes real-time sensor data, predicts the corresponding text for the gesture, and then converts it to speech, providing an interactive feedback experience.
